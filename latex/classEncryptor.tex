\doxysubsection{Подробное описание}
Шифрование методом маршрутной перестановки 

Ключ передается как аргумент в методах encrypt и decrypt. Для зашифровывания предназначен -\/ encrypt, для расшифровывания -\/ decrypt. \begin{DoxyWarning}{Предупреждения}
Реализация только для английского языка 
\end{DoxyWarning}
